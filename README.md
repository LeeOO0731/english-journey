# 🐰 My English Journey - 英语打卡

一个精美的英语学习打卡网页应用，采用Awwwards风格设计，粉白配色，融入美乐蒂元素。

## ✨ 功能特性

- **📅 打卡日历** - 可视化展示每月打卡记录，从2026年6月开始
- **📋 学习日程** - 添加、管理学习计划，支持标记完成状态
- **📊 数据统计** - 实时显示学习时长、打卡天数等统计
- **😊 心情打卡** - 记录每天的学习心情
- **☁️ 数据导出** - 支持导出JSON备份，可同步到坚果云、OneDrive等
- **💖 精美UI** - 玻璃态设计，柔和渐变，流畅动画

## 🚀 快速开始

直接在浏览器打开 `index.html` 即可使用！

## 📱 截图

- 打卡日历清晰展示每日打卡情况
- 日程表帮助规划学习
- 美乐蒂风格让打卡更有趣

## 💾 数据存储

所有数据保存在浏览器本地存储（localStorage）中，点击"同步打卡"按钮可导出备份。

## 🎨 设计风格

- 粉白主色调，美乐蒂风格
- 玻璃态卡片设计
